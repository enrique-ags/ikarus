Explaining crud operations(Create, Read, Update, Delete) on GoLang Language and MySQL simple way

Prerequisites for this sample:

  • Windows 10
  • MySQL
  • go version go1.16.5 windows/amd64

Creating sample datatabase and table in MySQL

Type commands below to create DB

MariaDB [directory]> create database directory;

MariaDB [directory]> use directory;

MariaDB [directory]> create table directory (name varchar(255), lastname varchar(255), age int, phone varchar(100) );

Explanation:
  • First line creates database directory
  • Second line is telling mysql to use directory db
  • Third line will create table directory with fields name, full name, age, int and phone

Manually Inserting some records in directory table MySQL

> insert into directory values('Jhon','Doe',45,'000-000-9999');

MariaDB [directory]> insert into directory values('Maria','Clayton',35,'000-000-9999');

MariaDB [directory]> insert into directory values('Robert','Smith',42,'000-000-9999');

MariaDB [directory]> insert into directory values('Gipsy','Kara',36,'000-000-9999');

MariaDB [directory]> insert into directory values('Mohamed','Reynolds',15,'000-000-9999');

Creating records using GoLang and MySQL(Insert)

Create Records using GoLang MySQL(MariaDB)
Steps
  1. Create folder directory
  2. move into -> 'directory' folder
  3. type in terminal: go mod init example.com/directory
  4. New file will be created-> go.mod
  5. Execute this command go get github.com/go-sql-driver/mysql
  6. Step above is to load the MySQL driver
  7. Create a file: directory.go in the same folder created above
  8. edit file: directory.go with the code below
    9. 
                        package main
                        
                        import (
                        
                            "database/sql"
                            _ "github.com/go-sql-driver/mysql"
                        )
                        func main() {
                            //database user is root
                            //empty password
                            db, err := sql.Open("mysql", "root:@tcp(127.0.0.1:3306)/directory")
                            
                            if err != nil {
                                panic(err.Error())
                            }
                            defer db.Close()
                            insert, err := db.Query("INSERT INTO directory VALUES ('Sam','Smith',4,'xxx-xxx-xxxx'),('Elliot','Gutierrez',55,'yyy-ddd-9999')")
                            
                            if err != nil {
                                panic(err.Error())
                            }
                        
                            defer insert.Close()
                        
                        }
  9. In order to compile and execute:
  10. type go install example.com/directory
  11. if no errors , type directoy (an exe file will be generated, some antivirus will show a warning)
  12. In MySQL terminal type
    13.                             MariaDB [directory]> select * from directory;
                            +--------+-----------+------+--------------+
                            | name   | lastname  | age  | phone        |
                            +--------+-----------+------+--------------+
                            | Sam    | Smith     |    4 | xxx-xxx-xxxx |
                            | Elliot | Gutierrez |   55 | yyy-ddd-9999 |
                            +--------+-----------+------+--------------+
  13. 2 records were inserted
